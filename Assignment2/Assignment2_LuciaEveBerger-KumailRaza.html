def create_class_sp():
    rates_of_survival = {'1': 0, '2': 0, '3': 0}
    rates_of_death = {'1': 0, '2': 0, '3': 0}
    for i in range(1, 4):
        rates_of_survival[str(i)] = \
            len(survival_rate.loc[survival_rate['pclass'] == i])/len(survival_rate)
        rates_of_death[str(i)] = \
            len(death_rate.loc[death_rate['pclass'] == i])/len(death_rate)
    death_chart = create_wedge_chart(rates_of_death, Greys, "Rate of Mortality by Class (Total Rate: {} %)".format(total_death_rate))
    survival_chart = create_wedge_chart(rates_of_survival, Blues, "Rate of Survival by Class (Total Rate: {} %)".format(total_survival_rate))
    show(row(survival_chart,death_chart))

def create_wedge_chart(data, colors, title):
    data = pd.Series(data).reset_index(name='value').rename(columns={'index': 'rates_of_death'})
    data["legend"] = "Economic Class: " + (data['rates_of_death'].astype(str)) + " (" + (round(data['value'],3).astype(str)) + "%)"                        
    data['angle'] = data['value'] / data['value'].sum() * 2 * pi
    data['color'] = colors[len(data)]
    p = figure(width=500, title=title, toolbar_location=None,
               tools="hover", tooltips="Class: @rates_of_death: @value %", height=350)
    
    p.wedge(x=0, y=1, radius=0.4,
            start_angle=cumsum('angle', include_zero=True), end_angle=cumsum('angle'),
            line_color="white", fill_color='color', legend='legend', source=data)
    p.title.align = 'center'
    p.axis.axis_label=None
    p.axis.visible=False
    p.grid.grid_line_color = None
    return p

def survival_rate_by_age():
    age_sorted = df.sort_values(by='age')
    columns = ['age_bracket', 'count', 'survival', 'male', 'female', 'male_percentage', 'female_percentage']
    lst = list()
    for i in range(1, 90, 10):
        subset = age_sorted.loc[(age_sorted["age"] < i + 10) & (age_sorted["age"] > i - 10)]
        count_of_male = len(subset.loc[
            (subset['sex'] == 'male') & (subset['survived'] == 1)])
        male_total = len(subset.loc[(subset['sex'] == 'male')])
        female_total = len(subset.loc[(subset['sex'] == 'female')])
        count_of_female = len(subset.loc[
                                (subset['sex'] == 'female') & (subset['survived'] == 1)])
        lst.append([i, len(subset), len(subset.loc[subset['survived'] == 1]) / len(subset),
                    count_of_male, count_of_female, count_of_male/male_total, count_of_female/female_total])
    
    age_survival_dict = pd.DataFrame(lst, columns=columns)
    return age_survival_dict
    
    
def render_graphs(data_input):

            'male': data_input['male'],
            'female': data_input['female'],

    if c_dict.get(c_name) == None:
        c_dict[c_name] = c_val
    else:
        c_dict[c_name] = c_dict.get(c_name) + c_val

from collections import defaultdict

def prepCabinData(df):

    df_home = df['cabin']
    n = np.array(df_home.get_values())
    num = 0
    for x in n:
        if isinstance(x, float):
            num +=1



    df_home_destinations = df['cabin'].value_counts()
    c_val = df_home_destinations.get_values()

    cabins = np.array(df_home_destinations.keys())
    c_name = []
    for val in cabins:
        c_name.append(val[0])

    c_dict = defaultdict(int)

    for nam,v in zip(c_name, c_val):
        checkInst(c_dict,nam,v)    
    c_dict['NaN'] = num
    return c_dict


def prepCabinSurvData(df):
    cs_data = df[df['survived'] == 1].groupby('cabin')['survived'].count()
    cs_val = np.array(cs_data.get_values())
    cs_names = np.array(cs_data.keys())
    cs_name = []

    for val in cs_names:
        cs_name.append(val[0])

    cs_dict = defaultdict(int)

    for cs_n, cs_val in zip(cs_name, cs_val):
        checkInst(cs_dict, cs_n, cs_val)
    return cs_dict

Titanic Dataset Findings:¶

Findings:¶

In the dataset, we found that majority of passengers were travelling to destinations in North America. When analyzing the survival rate, overall, the data showed a higher survival rate for first class passengers. Given that approximately 2/3 of the passengers died, third class passengers had a devestating mortality rate with 65%, where first class passengers fared much better with 15% mortality rate. Age and sex were fundamentally important in assessing the survival rate. Irrespective of sex, younger (0-10) and elderly (70+) saw much higher survival rates, both over 50%. Sex made a large difference, as the bar chart shows. Presumably, female passengers were evacuated first and thus their survival rates were always over 60%.

Also the survival rate was better for passengers having 1 or 2 family members on board (about 51% and 45% respectively). For the people having more than 5 family members on board, the survival rate is almost zero. Most of the infants, kids and teenagers were in the 3rd class. This also tells us that the 3rd class was quite populated. The percentage of young adults and middle aged people in the 1st class is almost the same. There were large number of young adults in the 2nd class. There were a small number of elderly, all of them were in the third class. The maximum survival percentage has been for the cabins E, B, D. The data doesn't include the cabin layout of the ship but one can infer that these cabins must have been near the exit doors and/or the escape rafts so the people there had early access. The cabin C was quite populated so the survival percentage is just 37%. Cabin T was an outlier where survival percentage was zero.

Post Mortem Analysis:¶

I found it difficult to draw conculsions on the survival rates as even the Third Class had 40%. Also the numbers fluctuate with the population size in different categories e.g one can infer that the number of infants in the third class was the largest, but it can simply be the case becuase the number of people travelling in the third class was the largest. It was difficult to draw intuitive conclusions from the correlations between some variables as they aren't seemingly related as we know them. Some data values were also missing from the dataset (e.g some passenger's cabin details) which made it hard to have a high confidence in the inference drawn.

On this particular dataset, it would be interesting to expand the data mining, perhaps develop a descision tree that could predict the survival rate on the discussed factors [age, sex, class, etc]. It is clear they are not independent and it could be interesting to see to what degree they influence each other. A multivariate analysis of the fields would also reveal some hidden aspects of the data which would allow us to understand (or estimate) the dynamic behavior/layout of the passengers which lead to their survival or death.
